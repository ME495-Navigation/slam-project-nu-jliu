# NUSLAM

**Author**: Allen Liu

This package is used for performing `Extended-Kalman-Filter` Based `SLAM` algorithm on a turtlebot

## Launch
To launch the slam in simulation, run
```
ros2 launch nuslam slam.launch.xml cmd_src:=teleop
```
